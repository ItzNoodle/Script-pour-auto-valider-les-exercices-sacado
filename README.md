💡 Purpose & Overview 

This script is designed to automatically process and validate multiple exercises on a page. 

It works by: 

Scanning the page for available exercise links 

Opening each exercise in the background 

Simulating realistic completion times 

Submitting the required form data automatically 

Displaying a live progress interface (status, progress bar, counter) 

The goal is to save time on repetitive actions while keeping behavior somewhat human-like. 

 

▶️ Usage Instructions 

⚠️ Important: You must be on the page that displays all the exercises (the list view with multiple items). The script will not work correctly on a single exercise page. 

Open the page that lists all exercises 

Press Ctrl + Shift + I (or F12) to open Developer Tools 

Go to the Console tab 

Paste the script below into the console 

Press Enter to run it 

A small interface will appear on the page 

Click “Démarrer” to start the automation 

Reload the page to see the results 

The script will then process each exercise and update the progress in real time.
